# Landing Page

A showcase landing page for the Laravel Auth Starter Kits project.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The landing page is configured for GitHub Pages deployment. The built files will be in the `dist` directory.

### GitHub Pages

1. Build the project: `npm run build`
2. Deploy the `dist` folder to GitHub Pages
3. Or use GitHub Actions for automatic deployment

## Features

- Interactive authentication demos
- Responsive design with Tailwind CSS
- Vue.js 3 with Composition API
- Fast development with Vite
